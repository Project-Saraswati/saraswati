# Code of Conduct

## Our Pledge
We are committed to creating a welcoming, inclusive, and respectful environment for everyone who participates in this project. We value collaboration, curiosity, and good-faith effort, regardless of background, experience level, or identity.

This project exists to improve transparency and equity in healthcare. Our conduct should reflect that mission.

---

## Our Standards
Examples of behavior that contributes to a positive environment include:

- Being respectful, thoughtful, and constructive in communication  
- Welcoming newcomers and supporting learning  
- Giving and accepting feedback gracefully  
- Focusing on whatâ€™s best for the project and its users  
- Assuming good intent and asking clarifying questions when something is unclear  

Examples of unacceptable behavior include:

- Harassment, discrimination, or exclusionary behavior of any kind  
- Personal attacks, insults, or derogatory comments  
- Trolling, bad-faith arguments, or deliberate disruption  
- Sharing private or sensitive information without permission  
- Any conduct that undermines trust, safety, or collaboration  

---

## Scope
This Code of Conduct applies to all project spaces, including but not limited to:

- GitHub repositories (issues, pull requests, discussions)  
- Slack or other communication channels related to the project  
- Any other space where contributors represent this project  

---

## Enforcement
Project maintainers are responsible for clarifying standards and enforcing this Code of Conduct fairly and consistently.

If unacceptable behavior occurs, maintainers may take actions they deem appropriate, including:

- Issuing a warning  
- Temporarily or permanently removing access to project spaces  
- Rejecting contributions that violate this Code of Conduct  

---

## Reporting
If you experience or witness behavior that violates this Code of Conduct, please report it privately to the project maintainers.


All reports will be reviewed and handled with care, discretion, and respect.
